arxiv.vault.middleware module¶

Provides middleware for getting secrets from Vault.

class arxiv.vault.middleware.VaultMiddleware(wsgi_app, config={})[source]¶

Bases: object

Middleware for populating Vault secrets on a request.

Config parameters:

  • KUBE_TOKEN, used to authenticate against the Kubernetes Auth endpoint.

  • VAULT_HOST

  • VAULT_PORT

  • VAULT_REQUESTS; see SecretsManager for how these should be expressed.

  • VAULT_SCHEME (optional; defaults to ‘https’)
